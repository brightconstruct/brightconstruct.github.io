Some people fill dozens - even hundreds - of forms weekly:

  • Teachers sending student forms

üí° "But isn't prefilled URL already a built-in Google Forms feature?"

Yes - technically.

And here's exactly where users get stuck.

To manually generate prefilled links at scale, you need to:

  • Extract hidden entry.xxxxx IDs
  • Build URL parameters by hand
  • Encode text (spaces, special chars, "&" breaking things)
  • Handle dropdown & checkbox logic
  • Split multiple selections
  • Maintain formulas when the form changes
  • Debug mismatches when values differ
  • Repeat for 100+ rows

Sure - spreadsheet wizards could hack something. But tools aren't about what's possible.

Good software turns "possible but painful" into easy, reliable, and accessible.

Software wins when it saves time, avoids mistakes, and removes complexity. Form Prefiller does exactly that.

üßµ Tech stack decisions & learnings

  • Google Apps Script -> easiest way to live inside Google Forms
  • OpenAI API -> natural for field matching
  • Script Properties -> track weekly AI usage
  • No external database -> user trust + simpler approval
  • Careful scope usage -> passing Google security review matters

Real-world problem forced "lean, compliant, user-friendly" thinking.

‚ö†Ô∏è Pitfalls I avoided (or learned the hard way)

  • ‚úÖ Respect Google API scope rules
  • ‚úÖ Never send user data off-Google
  • ‚úÖ No silent failures - always show warnings
  • ‚úÖ Make UX friendly for non-technical users
  • ‚úÖ Track free AI credits to prevent abuse
  • ‚úÖ Support dropdowns, checkboxes, time, etc. (not just text)

The hardest part: teaching users what "prefill" means in a clean, non-technical way.

üß≠ If you're building something similar

Find workflows where users repeat the same task + spreadsheets are involved.

Ask:

  • Who is doing repetitive manual work?
  • Why hasn't this been automated?
  • Can you build something simple but valuable first?
  • Will users trust you inside their workspace?

And the most important: Build for the 98% of people who don't know formulas or apps scripts - but feel the pain every day.
